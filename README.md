## S3

```bash
# list s3 buckets
aws s3 ls

# show details of s3 buckets
aws s3api list-buckets

#upload a file
aws s3 cp <SOURCE_FILE> <DESTINATION_FILE> s3://<BUCKET_NAME>
```
